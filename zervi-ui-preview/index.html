About This Preview

This is a static HTML preview of the Zervi MRP application user interface. The original application is built with React and Material UI, but this preview uses plain HTML to display the layout and design without requiring the full application stack.

Select a section below to view the UI components for that area of the application.

Dashboard

Main overview with key metrics and KPIs for each division

Manufacturing

Manufacturing orders, operations, and production scheduling

Inventory

Inventory management, stock levels, and warehouse locations

Divisions

Division-specific views for Automotive, Camping, Apparel, and Zervitek

Transfers

Inter-division transfer system for materials and semi-finished goods

Original Project Structure

The Zervi MRP application is built with the following technologies:

  • Frontend: React with Material UI components
  • Backend: Node.js with Express and TypeScript
  • Database: PostgreSQL with TypeORM

The application's frontend has a structure of reusable components and dedicated pages for each area of functionality.
